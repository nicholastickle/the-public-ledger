# Hero video asset

Place a file named `parliament-timelapse.mp4` here.

The video should be:
- Subject: UK Parliament / Palace of Westminster exterior
- Style: still camera, timelapse (clouds moving, daylight shifting)
- Aspect ratio: 16:9 or wider
- Length: 10–30 s loop (it loops automatically)
- No audio required (video is muted)

Free sources: Pexels or Pixabay both have suitable royalty-free clips.
Search "parliament timelapse" or "Westminster timelapse".

Recommended encoding: H.264 MP4, 1920×1080, ~5–15 MB.
